### 內容：
下載台灣政府資料開放平臺提供空氣品質指標(AQI)資料，列表顯示各地AQI數值和狀態。<br/>
使用JAVA寫成，主要練習項目如下：
1.  RecyclerView
2.  Progress Spinner
3.  SQLite
4.  Gson
5.  OkHttp

開放資料利用網址：
https://opendata.epa.gov.tw/Data/Details/AQI/?show=all
